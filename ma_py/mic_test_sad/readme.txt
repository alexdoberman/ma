Cистему тестов для VAD. Она аналогична системе для теста SDR
В качестве метрики используем F1 меру.

В итоге 
.\data\_vad_test - тут лежат тестовые данные
.\data\_vad_test\_speech+prodigy_-10dB\mix - wav файлы с решетки
.\data\_vad_test\_speech+prodigy_-10dB\config.cfg - конфиг описывающий данные
.\data\_vad_test\_speech+prodigy_-10dB\cut_speech.seg  - файл сегментации (полученный при помощи wave assistant) отмеченны участки речи.


\mic_test_vad - тут лежат скрипты запуска.  Сейчас там реализованно 3 VAD 
1) который все отмечает как речь
2) который все отмечает как  не речь
3) фазовый vad
---------------------------------------------------------------------------------------------------------------------------------------------

Для работы необходимо поставить пакет 
pip install pyinterval
---------------------------------------------------------------------------------------------------------------------------------------------


Для теста алгоритма VAD пишем наследника от BaseVAD (base_vad.py)


В результате алгоритм  должен отдать разметку в формате 
np.array([[time_begin, time_end], .... [time_begin, time_end]])

time_begin - время начала речи
time_end  - время окончания речи
